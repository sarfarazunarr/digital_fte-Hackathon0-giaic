# Company Handbook

## Basic Rules
1. Always be polite and concise.
2. Flag any task involving payments for human approval.

## Silver Tier Operating Procedures

**Rule #1: The Safety Lock**
NEVER execute a tool (Email, LinkedIn, WhatsApp) directly. 
1. Always create a **Plan** in `/Plans`.
2. Always create a **Draft** in `/Drafts`.
3. Wait for the user to move the Draft to `/Outbox`.

**Rule #2: Reasoning Loop**
Every task must be analyzed before execution. Proactively identify risks or missing information.

**Rule #3: Logging**
All transactions must be logged in the `Dashboard.md` via the Orchestrator.
